# 💎 BRI JOYERÍA - ERP de Gestión de Inventario

¡Bienvenido a **BRI JOYERÍA**! Este es un sistema integral de gestión (ERP) diseñado para administrar el inventario de una joyería de alta gama de manera eficiente, elegante y segura.

## 🚀 Características principales
* **Gestión de Inventario (CRUD):** Registro, edición, visualización y eliminación de joyas.
* **Cálculo de Capital:** Visualización en tiempo real del valor total del inventario en bóveda.
* **Galería Visual:** Soporte para imágenes de productos para una identificación rápida.
* **Diseño Premium:** Interfaz moderna con tonos dorados y oscuros, optimizada para la experiencia del usuario.

## 🛠️ Tecnologías utilizadas

### Frontend:
* **React.js**: Biblioteca principal para la interfaz de usuario.
* **Vite**: Herramienta de construcción rápida para el frontend.
* **Axios**: Para la comunicación fluida con la API.
* **Lucide React**: Iconografía elegante.

### Backend:
* **ASP.NET Core (C#)**: Motor del servidor y API RESTful.
* **Entity Framework Core**: ORM para la gestión de la base de datos.
* **SQL Server**: Base de datos robusta para el almacenamiento de productos.

## 📋 Requisitos previos
1.  Tener instalado **Node.js** (para el frontend).
2.  Tener instalado **.NET SDK 8.0** o superior (para el backend).
3.  Instancia de **SQL Server** activa.
